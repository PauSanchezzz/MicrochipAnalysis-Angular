Animales identificados con Microchip por localidad en Bogota D.C.

Este proyecto tiene como objetivo analizar los datos relacionados con animales que cuentan con microchips en diversas localidades de Bogotá. Estos datos fueron extraídos de los conjuntos de datos abiertos proporcionados por el Gobierno Digital de Colombia, los cuales consisten en información pública disponible en formatos que permiten su uso y reutilización bajo licencia abierta, sin restricciones legales para su aprovechamiento.

En el desarrollo de este proyecto, se llevó a cabo una limpieza de los datos descargados. Posteriormente, se creó la base de datos en PostgreSQL mediante el ORM SQLAlchemy. Luego, se procedió a la extracción de la información y su almacenamiento en la base de datos de PostgreSQL usando Python. Finalmente, se llevó a cabo la visualización de estos datos utilizando el framework Angular, en conjunto con Tailwind y Python para la generación de gráficas.
